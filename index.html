Skeleton Resumé

During this assignment, I learn how to code using Skeleton Framework. First, I built my resume using skeleton 12-column fluid grid. Next, I made media queries to support the display of different screen sizes. Finally, I link my resume with my home page. In conclusion. I learn a lot from skeleton framework. For example, I learn how to put multiple items inside a container. You can combine images and paragraph with different positions inside a container. You can also abject their placement with rows and column. I like how the CSS had already existing code. This made it easy for me to create. The hard part about this assignment was familiarizing myself with the CSS code.

Flexbox

In this assignment, I learn about the Flexbox Framework. The Flexbox Framework showed me how to create a responsive website. During this assignment, I follow steps on how to create responsive webpages like the one shown on my portfolio. Flexbox makes creating website easy, for different-sized devices. I like how easy its’ uses of column and row are. There were not much I didn’t like about Flexbox. Following the steps to create the website was fun and simple, I would definitely use Flexbox for future projects.

Bootstrap

Bootstrap is an excellent framework use to create a responsive website. This shows that, Bootstrap provides responsive web inputs and uses a mobile-first approach. Bootstrap comes with a variety of templates such as buttons, tables, and navigation. Using Bootstrap allows for quick development of web pages. Bootstrap’s media queries made it easy and quick to create a responsive design. What I didn’t like about Bootstrap was familiarizing myself with the pre-install codes, other than that, Bootstrap is the ideal framework to use for beginners.
